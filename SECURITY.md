# Security

This is a personal portfolio repository. The backend ships hard-coded
demo credentials and an in-memory store on purpose — it exists to back
the API test suite, not to be deployed.

If you find a real issue (a published secret, a dependency advisory I
missed, a backend that someone could leverage off-portfolio), email
**shanaleks0007@gmail.com**. I'll acknowledge within 48h.

Please don't open a public issue for anything that looks sensitive.
